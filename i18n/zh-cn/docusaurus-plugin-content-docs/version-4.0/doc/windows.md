# Windows下的SRS

SRS现在有Windows版本了，是基于cygwin64实现的，请参考[srs-windows](https://github.com/ossrs/srs-windows)。

当然，可以使用Docker运行SRS，请使用[ossrs/srs:dev](https://github.com/ossrs/dev-docker/tree/dev#usage)。
