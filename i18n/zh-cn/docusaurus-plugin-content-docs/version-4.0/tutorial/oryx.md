# Oryx合集

Oryx(SRS Stack)，面向鼠标编程，没有服务器和命令行经验，也可以构建自己的在线视频服务。入门的最佳推荐。

Oryx是一个开源的音视频服务方案，多个音视频场景中开箱即用，组合使用了SRS、FFmpeg、WebRTC等项目，无门槛赋能小微企业，让小微企业轻松具备数字化能力。

Oryx：让音视频的力量，赋能所有小微企业。
